Formato del archivo CSV

Se aceptan archivos CSV exportados desde la web o compatibles con el formato clínico de importación.

Columnas obligatorias:
Edad, Sexo/Femenino, Capnometria, Causa_cardiaca o Causa_fallecimiento_codigo, Cardio_manual, Recuperacion_pulso, Prediction_mode

También se aceptan variantes de nombres:
• Sexo o Femenino.
• Cardiocompresion, Cardiocompresión o Cardio_manual.
• Capnometria o Capnometría.
• Recuperacion_pulso, Recuperación_pulso o Rec. del pulso.

Reglas:
• Sexo/Femenino acepta Si/No, 1/0, true/false, Hombre/Mujer o Masculino/Femenino.
• Causa_cardiaca y Recuperacion_pulso aceptan Si/No, 1/0 o true/false.
• Cardio_manual debe ser Manual o Mecánica.
• Prediction_mode debe ser MID_RCP o AFTER_RCP.

Campos opcionales del formato web/ML:
IMC, Grupo_sanguineo, Adrenalina, HTA, Diabetes, Tabaco, Colesterol, Alcohol, Resultado_ML, Probabilidad_ML y datos del modelo.

Notas:
• MID_RCP corresponde a la predicción a los 20 minutos de RCP.
• AFTER_RCP corresponde a la predicción después del procedimiento de RCP.
• Las filas duplicadas no se importarán.
• Si falta una columna obligatoria o algún valor no es válido, la fila será descartada.
