Formato del archivo CSV

Solo se acepta el CSV exportado desde la web.

Columnas obligatorias:
Edad, Femenino, Capnometria, Causa_cardiaca, Cardio_manual, Recuperacion_pulso, Prediction_mode

Reglas:
• Femenino, Causa_cardiaca y Recuperacion_pulso deben ser Si/No.
• También se aceptan 1/0, true/false, Hombre/Mujer o Masculino/Femenino.
• Cardio_manual debe ser Manual o Mecánica.
• Prediction_mode debe ser MID_RCP o AFTER_RCP.

Notas:
• MID_RCP corresponde a la predicción a los 20 minutos de RCP.
• AFTER_RCP corresponde a la predicción después del procedimiento de RCP.
• Si falta alguna columna obligatoria o algún valor no es válido, el archivo no se importará.
